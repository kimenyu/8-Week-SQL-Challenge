# 8-Week-SQL-Challenge is a challenge created by Danny ma. The challenge can be accessed here <a href = "https://8weeksqlchallenge.com"> 8 week sql challnmege </a>.

Health analytics
Marketing analytics
People analytics
Financial markets
Fast moving consumer goods
Digital marketing
